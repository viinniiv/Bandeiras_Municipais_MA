# Bandeiras Municipais do Maranhão (MA)

Este repositório reúne um acervo de imagens das bandeiras dos municípios do estado do Maranhão (MA), organizado para facilitar uso em projetos, pesquisas, materiais educacionais e aplicações.

- As bandeiras estão vinculadas ao código IBGE de cada município, permitindo identificação padronizada e fácil integração com bases de dados oficiais.
